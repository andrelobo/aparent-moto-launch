# Aparent Moto Launch

Este é um aplicativo web React para um serviço de aluguel de motocicletas chamado "Aparent Moto Launch". É um site de marketing de página única que apresenta os serviços da empresa, preços e informações de contato.

## Tecnologias

-   **Framework:** React
-   **Ferramenta de Build:** Vite
-   **Linguagem:** TypeScript
-   **Estilização:** Tailwind CSS com componentes shadcn/ui
-   **Roteamento:** React Router DOM

## Desenvolvimento

Para iniciar o projeto, siga estes passos:

### 1. Instalação

Instale as dependências do projeto usando `yarn`:

```bash
yarn install
```

### 2. Rodando o Servidor de Desenvolvimento

Para iniciar o servidor de desenvolvimento local, execute:

```bash
yarn dev
```

O aplicativo estará disponível em `http://localhost:5173` (ou na próxima porta disponível).
